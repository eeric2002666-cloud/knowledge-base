# knowledge-base

基于 Karpathy 的 LLM Wiki 模式构建的个人知识库管理系统。

## 功能

- 📥 一键添加链接/文件/笔记/对话结论
- 🔍 搜索和回顾
- 💬 保存与 AI 对话的结论
- 🔄 增量编译持久维基百科
- ✨ Write-back：知识通过使用增长
- 🏥 健康检查：定期检查知识库健康度

## 安装

```bash
npx skills add 你的GitHub用户名/knowledge-base
```

## 使用

| 操作 | 指令 |
|------|------|
| 添加链接 | `把这个加到知识库：https://...` |
| 添加结论 | `记录这个结论：...` |
| 搜索 | `搜索知识库 xxx` |
| 回顾 | `本周新增了什么` |
| 整理 | `整理知识库` |
| 健康检查 | `知识库健康检查` |

## 架构

基于 Karpathy 的 LLM Wiki 模式：
- **Sources**: 原始源（不可变）
- **Wiki**: LLM 编译的互联页面
- **Schema**: 页面格式和命名惯例
